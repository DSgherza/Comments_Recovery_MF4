# LoLa_project
Longitudinal and Latarel analysis tool. 

The modules installation string is:
'python -m pip install --trusted-host pypi.org --trusted-host pypi.python.org --trusted-host files.pythonhosted.org --upgrade hydra-core'

If you want to install a module, you must substitute hydra-core with the module name.

If you want to activate a virtual environment for your current user and you have no administrator privileges, you have to execute the command "Set-ExecutionPolicy -Scope CurrentUser RemoteSigned" in the terminal and then execute the command ".\test_env\Scripts\Activate.ps1"
